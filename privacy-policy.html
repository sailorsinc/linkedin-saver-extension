Privacy Policy

LinkedIn Saver is a browser extension that allows users to save LinkedIn profiles as resume-style PDFs for personal use.

Data Collection

This extension does not collect, store, or transmit any personal data. All processing is performed locally in your browser. No data is sent to any server or third party.

How the Extension Works

  • The extension only accesses the content of the LinkedIn profile page you are viewing, and only when you click the extension button.
  • No information is collected or shared with the developer or any third party.
  • The generated PDF is saved directly to your device and is not uploaded anywhere.

Permissions

  • scripting: Used to inject code for extracting profile data and generating the PDF, only when triggered by the user.
  • activeTab: Used to run the extension only on the active tab when the user clicks the button.
  • downloads: Used to save the generated PDF to your device.

Contact

If you have any questions about this privacy policy, please contact the developer via the Chrome Web Store listing.
